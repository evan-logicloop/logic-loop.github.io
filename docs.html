Getting started



LogicLoop empowers your Operations team. Monitor and manage your business processes with rules to alert, review and act. Write your first rule in 5 mins with three easy steps:

1. Connect your data source.
2. Write a SQL query to express your rule.
3. Add an action to be triggered based on the results of your query.
View Tutorial

Create an account


Reach out to hi@getlogicloop.com to get started with an account. We respond within 24 hours. Self-serve account creation coming soon!

Connect your data sources


First, starting by connecting your data to LogicLoop. Add a new data source from the Settings > Data Sources tab. We support several relational, non-relational and API-based data sources. See a full list of data sources here.


Learn more about how to connect your data sources

Write a rule query


LogicLoop lets you write rules on your data to manage your business processes. Add a new rule from the Rules tab. You’ll write a SQL query, set up a schedule, and actions on the rule.


Learn more about how to write a rule

Add actions


LogicLoop lets you automate rules by monitoring and taking actions for you. To add an action to a rule, first create an Action Destination from the Destinations tab. We support several common actions such as alerts to Slack, email, PagerDuty and Zapier. You can also trigger custom actions via webhooks.


Learn more about how to set up actions



That’s all you need to get started with LogicLoop! You can always check how your business is doing by checking out the status of your rules on the Rules tab. You can read about some more advanced features here. Contact us at hi@getlogicloop.com if you have any questions.
