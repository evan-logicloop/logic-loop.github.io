Connecting your Data

LogicLoop supports several relational, non-relational and API-based data sources out of the box. See a full list of Data Sources here.

Note: If you need to connect to your database behind a VPC via SSH tunneling through a bastion host, or want to set up an on-prem solution, contact hi@getlogicloop.com and we can guide you through the setup.

Navigate to the Settings > Data Sources tab



Click ‘+ New Data Source’ to open the Create a Data Source wizard



Select the type of your data source, configure it, and click test connection to verify the setup



You can always edit a data source by clicking on it from the Data Sources listing page



Data sources connected! Next, use your data sources to write a rule.
