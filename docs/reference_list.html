Supported Action Parameters


These are parameters that can be embedded into custom action notification templates within curly braces. Click here to learn more about Actions.



ACTION_STATUS - The evaluated action status (string).
ACTION_CONDITION - The action condition operator (string).
ACTION_THRESHOLD - The action threshold (string or number).
ACTION_NAME - The action name (string).
ACTION_URL - The action page url (string).
RULE_NAME - The corresponding rule name (string).
RULE_URL - The corresponding rule page url (string).
RULE_RESULT_VALUE - The rule query result value (string or number)
RULE_RESULT_ROWS - The rule query result rows (value array)
RULE_RESULT_COLS - The rule query result columns (string array)
