Write a Rule

You’ll write rules to manage your business processes with LogicLoop. Before you write a rule, make sure to connect some data sources.

Navigate to the Rules tab. Here you see a listing of all existing Rules with useful metadata and the status of their last runs.



Click ‘+ New Rule’ to start writing a rule query

  • • Add a Name
  • • Select a connected data source and search the schema
  • • Write your SQL query, click ‘View Results’ to check it looks good, and ‘Save’. You can make changes and re-run to fine-tune your results.
  • • Publish your rule.
  • • (Optional) Add Tags, a Rule Description and other helpful documentation
  • • (Optional) You can run your rule on a Schedule. We support ‘Never’, ‘Minutes’ (1-30), ‘Hours’, ‘Days’, ‘Weeks’




You can run this rule as-is, but you’ll want to set up alerts and actions next.
